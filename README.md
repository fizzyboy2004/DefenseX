# DefenseX


# 🛡 DefendX – Cybersecurity Learning Game

🎮 An interactive web-based game to simulate real-world cyber attack defense and teach network security concepts.

🌐 **Live Demo:**
https://fizzyboy2004.github.io/DefenseX/

---

## 🚀 Features

* 📊 Traffic Monitoring
* ⚡ Rate Limiting
* 🔥 Firewall Simulation
* 🚫 IP Blacklisting
* ⚖ Load Balancing
* 🔐 Brute Force Protection

---

## 🧠 About the Project

DefendX is designed to make cybersecurity learning **simple, interactive, and engaging**.
Users play as defenders and protect a server from different types of cyber attacks.
